## Reconnaissance Frameworks

These frameworks aim to provide a complete suite of tools for web reconnaissance:

| Tool           | Description |
|---------------|------------|
| **FinalRecon** | A Python-based reconnaissance tool offering various modules for tasks like SSL certificate checking, Whois information gathering, header analysis, and crawling. It has a modular structure for easy customization. |
| **Recon-ng**   | A powerful Python-based framework with a modular structure, supporting tasks such as DNS enumeration, subdomain discovery, port scanning, web crawling, and exploiting known vulnerabilities. |
| **theHarvester** | A command-line tool written in Python, designed for gathering email addresses, subdomains, hosts, employee names, open ports, and banners from public sources like search engines, PGP key servers, and the SHODAN database. |
| **SpiderFoot** | An open-source intelligence automation tool that integrates with various data sources to collect information about a target, including IP addresses, domain names, email addresses, and social media profiles. It also performs DNS lookups, web crawling, and port scanning. |
| **OSINT Framework** | A collection of various tools and resources for open-source intelligence gathering, covering social media, search engines, public records, and more. |

### FinalRecon

**FinalRecon** offers a wealth of reconnaissance information:

- **Header Information**: Reveals server details, technologies used, and potential security misconfigurations.
- **Whois Lookup**: Uncovers domain registration details, including registrant information and contact details.
- **SSL Certificate Information**: Examines the SSL/TLS certificate for validity, issuer, and other relevant details.
- **Crawler**:
  - **HTML, CSS, JavaScript**: Extracts links, resources, and potential vulnerabilities from these files.
  - **Internal/External Links**: Maps out the website's structure and identifies connections to other domains.
  - **Images, robots.txt, sitemap.xml**: Gathers information about allowed/disallowed crawling paths and website structure.
  - **Links in JavaScript, Wayback Machine**: Uncovers hidden links and historical website data.
- **DNS Enumeration**: Queries over 40 DNS record types, including DMARC records for email security assessment.
- **Subdomain Enumeration**: Leverages multiple data sources (*crt.sh, AnubisDB, ThreatMiner, CertSpotter, Facebook API, VirusTotal API, Shodan API, BeVigil API*) to discover subdomains.
- **Directory Enumeration**: Supports custom wordlists and file extensions to uncover hidden directories and files.
- **Wayback Machine**: Retrieves URLs from the last five years to analyze website changes and potential vulnerabilities.
