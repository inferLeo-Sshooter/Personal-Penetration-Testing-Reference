
## DNS

| Component | Description |
|-----------|-------------|
| **DNS Purpose** | Resolves domain names (e.g., hackthebox.com) to IP addresses without central database |
| **Server Types** | DNS root, Authoritative/Non-authoritative name servers, Caching, Forwarding, Resolver |
| **Distribution** | Globally distributed across thousands of name servers like distributed phone books |
| **Security Risk** | Mainly unencrypted; vulnerable to local WLAN and ISP interception |
| **Encryption Solutions** | **DoT**: DNS over TLS<br>**DoH**: DNS over HTTPS<br>**DNSCrypt**: Computer-to-nameserver encryption |
| **Additional Data** | Stores service information beyond IP mapping (email servers, name servers, etc.) |
