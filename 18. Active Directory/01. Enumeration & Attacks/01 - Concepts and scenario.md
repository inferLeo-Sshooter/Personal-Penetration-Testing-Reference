

# Introduction

## 1. Active Directory Explained

Active Directory (AD) is a directory service for Windows enterprise environments, officially launched in 2000 with Windows Server 2000. Based on x.500 and LDAP protocols, AD is a distributed, hierarchical system enabling centralized management of organizational resources—including users, computers, groups, network devices, file shares, group policies, and trusts. AD provides authentication, accounting, and authorization functions within Windows enterprise environments.
