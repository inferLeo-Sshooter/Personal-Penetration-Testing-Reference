# 1. Filter/WAF Detection

For example, we are usign Host Checker web application to testing around. We send our payload: `127.0.0.1; whoami`. However, we get the error message `invalid input`. 

Security mechanism blocked our request. Error appears in the output field, indicating detection by the PHP application itself.

**Note:** If a different page appears showing our IP and request details, this likely means a Web Application Firewall (WAF) intercepted it instead.

Other than the IP (which we know is not blacklisted), we sent:
- A semi-colon character `;`
- A `space` character
- A `whoami` command

So, the web application either detected a blacklisted character or detected a blacklisted command, or both. There for, we will try figure out what is blacklisted by sending the correct input (`127.0.0.1`) with each of element above. (`127.0.0.1;`, `127.0.0.1 `, `127.0.0.1whoami`).
