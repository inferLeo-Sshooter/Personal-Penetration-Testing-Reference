# Linux (Bashfuscator)

Bashfuscator is a useful tool for bash command obfuscation that can help with security testing.

## Installation

```bash
git clone https://github.com/Bashfuscator/Bashfuscator
cd Bashfuscator
pip3 install setuptools==65
python3 setup.py install --user
```

## Basic Usage

Navigate to the bin directory:
```bash
cd ./bashfuscator/bin/
```

View help menu:
```bash
./bashfuscator -h
```

## Sample Command Obfuscation

Basic obfuscation (random technique):
```bash
./bashfuscator -c 'cat /etc/passwd'
```
Output: A potentially very long obfuscated command

## Creating Shorter Obfuscated Commands

Use flags to create more manageable commands:
```bash
./bashfuscator -c 'cat /etc/passwd' -s 1 -t 1 --no-mangling --layers 1
```

Output:
```bash
eval "$(W0=(w \  t e c p s a \/ d);for Ll in 4 7 2 1 8 3 2 4 8 5 7 6 6 0 9;{ printf %s "${W0[$Ll]}";};)"
```

## Testing the Obfuscated Command

Verify the obfuscated command works correctly:
```bash
bash -c 'eval "$(W0=(w \  t e c p s a \/ d);for Ll in 4 7 2 1 8 3 2 4 8 5 7 6 6 0 9;{ printf %s "${W0[$Ll]}";};)"'
```

The command successfully executes `cat /etc/passwd` while appearing completely different from the original command, helping to evade command filters or detection.

# Windows (DOSfuscation)

DOSfuscation is an interactive tool for Windows command obfuscation similar to Bashfuscator but designed for the Windows command line environment.

## Installation

```powershell
git clone https://github.com/danielbohannon/Invoke-DOSfuscation.git
cd Invoke-DOSfuscation
Import-Module .\Invoke-DOSfuscation.psd1
```

## Basic Usage

Launch the interactive interface:
```powershell
Invoke-DOSfuscation
```

View available options:
```
Invoke-DOSfuscation> help
```

## Sample Command Obfuscation

Set the command you want to obfuscate:
```
Invoke-DOSfuscation> SET COMMAND type C:\Users\htb-student\Desktop\flag.txt
```

Choose an obfuscation technique:
```
Invoke-DOSfuscation> encoding
Invoke-DOSfuscation\Encoding> 1
```

Output:
```
typ%TEMP:~-3,-2% %CommonProgramFiles:~17,-11%:\Users\h%TMP:~-13,-12%b-stu%SystemRoot:~-4,-3%ent%TMP:~-19,-18%%ALLUSERSPROFILE:~-4,-3%esktop\flag.%TMP:~-13,-12%xt
```

## Testing the Obfuscated Command

Verify the obfuscated command works in CMD:
```cmd
C:\htb> typ%TEMP:~-3,-2% %CommonProgramFiles:~17,-11%:\Users\h%TMP:~-13,-12%b-stu%SystemRoot:~-4,-3%ent%TMP:~-19,-18%%ALLUSERSPROFILE:~-4,-3%esktop\flag.%TMP:~-13,-12%xt
test_flag
```

## Tip
You can run DOSfuscation on Linux using PowerShell Core (`pwsh`). Simply install `pwsh` on your Linux system and follow the same steps as above.
